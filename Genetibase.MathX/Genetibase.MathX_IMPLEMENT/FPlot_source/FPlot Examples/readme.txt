Here are some example files you can use with FPlot.

- Gauss data.txt is an ASCII File with some sample measurement data, you can load into FPlot, in the 
  Edit menu -> Add data...

- Fractals.cs is a C# library file you can load as a Library in the Edit menu -> Edit library.
  After you have loaded and compiled the library you can refer to the library in a function with the
  following code:

  return Fractals.Mandelbrot.f(x, y);

- gaussfit.fplot is an example of some data fitted with three gauss curves.

- mandelbrot.fplot is an example of the Mandelbrot set.
