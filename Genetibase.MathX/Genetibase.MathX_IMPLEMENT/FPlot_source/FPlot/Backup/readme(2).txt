FPlotLibrary

FPlotLibrary is a library that implements a System.Windows.Form Control that displays mathematical functions
measurement data. The documentation for FPlotLibrary is in the file doc\FPlotLibrary.chm.

History:

FPlot 1.03:
Bugfix: Zoom beyond maximum zoom caused an exception
Bugfix: DataItem drawing was not completely threadsafe.

FPlot 1.04:
Bugfix: Added ICSharpZipLib to the project.

FPlot 1.05:
Bugfix: Drawing repeated in an endless loop when the scale width changed.
