Use layout algorithms

These samples describe how you can lay out a diagram.

The following code examplifies how you can create a random tree based on a utility function inside the Layout namespace. Via the static LAyoutSettings class you set various properties of the layout algorithms. Finally, the Layout method of the diagram control will lay out the nodes with the given layout algorithm. Easy, isn't it?

Note also the TreeOrientation enumeration; you can lay out the nodes in four directions without any extra coding.
