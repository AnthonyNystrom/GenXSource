Math.NET Symbolics: Yttrium - Package Overview

Yttrium is an experimental computer algebra architecture, implementing ideas and concepts of formal hardware engineering and digital information engineering, looking at abstract math and algebra from a different, new angle.

Yttrium is connected to the other new Math.NET packages like Numerics, PreciseNumerics and SignalProcessing, and thus takes direct advantage of the optimized numerical routines.

Math.NET Project News Blog with current ideas, thoughts and developments around the Math.NET project.

Complete the Core Architecture

Most of the Yttrium Core are designed and work fine, the rest es being completed, simplified and optimized.

Extending the Standard Library

The standard library (automatically loaded) covers basic data structures like integers, floating point numbers, modular rings as well as logic states. Additionally there are architectures for elementary operations of topics like logic, arithmetic, algebra and analysis/calculus. It's not certain yet whether topics like linear algebra or probability theory and models will be included into the standard library or (more likely) will be offered in seperate modules.

User Interface

Math.NET is a software library to be used in mathematical applications, but is not exactly such an end-user application. Nevertheless, Yttrium will provide several user interfaces just like Classic did. One of those is already working: a graphical frontend based on the Netron Graph Library.
